# Template Persona

## 1. Discovery Prompts

- **Guideline 1**: Instruct the discovery agent on what specific domain requirements to gather here.
- **Guideline 2**: Add another prompt instruction.

## 2. Specification Template

- **New Section Name**: Describe what should be injected into the final specification schema.

## 3. Auditing Rules

- **Rule 1**: Define a condition that should raise a `Blocker`.
- **Rule 2**: Define a condition that should raise a `Warning` or `Suggestion`.

## 4. Tone & Style

- Detail the communication style and tone the agent should adopt.

## 5. Anti-Patterns

- Specify topics or actions the agent must explicitly avoid.

## 6. Knowledge Context

- **Term 1**: Definition of the term to prevent the agent from asking the user.
